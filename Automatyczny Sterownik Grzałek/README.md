# Automatyczny Sterownik Grzałek (ASG) wersja 2.0

Projekt sterownika grzałek do teleskopu.

## Funkcje:
 + Pomiar temperatury
 + Pomiar wilgotności
 + Obliczenie punktu rosy
 + Dwa kanały PWM

## Wymagane Biblioteki:
 + Adafruit Unified Sensor (Adafruit)
 + DHT Sensor Library (Adafruit)
